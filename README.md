# Escenario 3D Navideño con Navegación Pointer Lock

Proyecto interactivo desarrollado con **Three.js** que presenta un escenario 3D con temática navideña, el cual puede explorarse mediante navegación en primera persona utilizando **Pointer Lock Controls**. El usuario puede interactuar con objetos específicos del entorno mediante **hover** y **click**, activando transiciones suaves de cámara hacia distintos puntos del escenario.

---

## Tecnologías utilizadas

* **JavaScript**
* **Three.js**
* **Pointer Lock Controls**
* **HTML5**
* **WebGL**

---

## Interacción y navegación

El proyecto implementa una experiencia inmersiva en primera persona con las siguientes características:

* Navegación en primera persona mediante **Pointer Lock**
* Movimiento de cámara controlado por el **mouse**
* Detección de interacción con objetos mediante **hover** y **click**
* Transiciones de cámara suavizadas entre puntos específicos del escenario
* Exploración guiada a través de elementos interactivos

---

## Funcionalidades principales

* Explorar puntos específicos del escenario 3D
* Interactuar con objetos al pasar el cursor sobre ellos
* Activar desplazamientos de cámara al hacer clic en elementos interactivos
* Cambiar de ubicación dentro del escenario de forma fluida y comprensible
